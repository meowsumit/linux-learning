# Linux Basics

## 1. Introduction to Linux Terminal
The Linux terminal is a powerful way to interact with your system.

## 2. Basic Commands

### ls
- **Description**: List files and directories in the current directory.
- **Example**: `ls`

### cd
- **Description**: Change directory.
- **Example**: `cd Documents`

### pwd
- **Description**: Print the current working directory.
- **Example**: `pwd`

### mkdir
- **Description**: Create a new directory.
- **Example**: `mkdir new_folder`

### rmdir
- **Description**: Remove an empty directory.
- **Example**: `rmdir old_folder`

### touch
- **Description**: Create a new empty file.
- **Example**: `touch file.txt`

### cp
- **Description**: Copy files or directories.
- **Example**: `cp source.txt destination.txt`

### mv
- **Description**: Move or rename files or directories.
- **Example**: `mv oldname.txt newname.txt`

### rm
- **Description**: Remove files or directories.
- **Example**: `rm file.txt`

### cat
- **Description**: Display the content of a file.
- **Example**: `cat file.txt`

### man
- **Description**: Show the manual for a command.
- **Example**: `man ls`

### clear
- **Description**: Clear the terminal screen.
- **Example**: `clear`

## 3. Conclusion
Learning these basic commands will help you use the Linux terminal more effectively. Remember to practice!
